# Security Policy

## Supported Versions

Security updates are applied to the latest `main` branch state.

## Reporting a Vulnerability

Please report security issues privately by emailing:
`megamaggi@gmail.com`

Include:

- A clear description of the issue
- Steps to reproduce
- Potential impact
- Any suggested remediation

You will receive an acknowledgement as soon as possible, and we will coordinate a fix and disclosure timeline.
